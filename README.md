## Random Walker 
Inspired by CodingTrain on Youtube

### Dependicies
- [SDL2](https://www.libsdl.org)
- ```make``` (optional)

### Building 
```
git clone https://github.com/mustardfrog/random-walker
cd random-walker
# if you have make installed
make && ./main

# without make
chmod +x build.sh 
./build.sh && ./main
```
